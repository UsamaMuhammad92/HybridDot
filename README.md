# HybridDot

Modern Gen-Z style creative, marketing, and technology agency website. Based in Lahore.

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** (design system: primary red, black, white, Space Grotesk + Inter)
- **Framer Motion** (animations)
- **React Three Fiber / Three.js** (3D-ready via `SceneCanvas`)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project structure

See **[ARCHITECTURE.md](./ARCHITECTURE.md)** for folder structure, key files, design system, and how to add 3D sections.

## Scripts

- `npm run dev` – Development server
- `npm run build` – Production build
- `npm run start` – Start production server
- `npm run lint` – ESLint
